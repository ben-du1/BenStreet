A React application with a Python backend that allows anonymous users to invest my money in dollar increments. This project uitlizes Interactive Brokers' Client Portal API for all of it's investment functionality.
<br />

Features:
- Realtime portfolio information
- Recent trade history
- Buy & sell stocks by ticker

<br />

Setup:
1. Install neccessary NPM and pip packages (see `requirements.txt` & `package.json`)
2. Download and run the Client Portal API from Interactive Brokers' website
3. Run `main.py` & `npm start`

<br />

Future Additions:
- Options Trading and Margin Accounts 🤑🤑
